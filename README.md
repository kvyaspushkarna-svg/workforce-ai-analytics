Workforce AI Analytics: HR Risk Assessment & Upskilling Platform
Overview:
This project provides a data-driven framework for assessing workforce automation risks and identifying strategic upskilling opportunities. It integrates generative AI with traditional data analytics to deliver actionable HR insights through a fully deployed interactive application.

Key Components:
1. Data Engineering: Engineered a comprehensive 500-employee dataset using Python, simulating complex organizational structures, performance variables, and department-specific metrics.
2. AI Intelligence: Leveraged the Claude and Gemini APIs to perform deep-dive analysis on role-based automation threats and generate tailored professional development roadmaps.
3. Executive Visualization: Designed a Tableau dashboard focused on a high-level Action Matrix, enabling leadership to pinpoint workforce vulnerabilities and allocate training resources with precision.
4. Production Deployment: Implemented version control via Git and deployed the final web application using Streamlit for public accessibility.

Technical Stack:
Languages: Python (Pandas, NumPy)
Generative AI: Claude API, Gemini API, Notebook LLM
Data Visualization: Tableau
Infrastructure: Streamlit, Git/GitHub

Strategic Value:
The system transforms raw workforce data into a predictive tool, allowing organizations to proactively manage the impact of AI on their human capital while fostering a culture of continuous, targeted learning.
